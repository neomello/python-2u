# 🐍 NEØ Protocol // Python Message APP

![NEØ Logo](images/neo-smart-factory-horizontal.png)

> **"Communication between humans would be much clearer if it were code written in Python."**

Uma interface minimalista e futurista que transforma suas mensagens em código Python puro e elegante. Desenvolvido sob os padrões do **NEØ Protocol**, com estética glassmorphism e foco total na experiência do desenvolvedor.

## ✦ Features

- **Real-time Conversion**: Digite em linguagem humana, receba em Python.
- **Multiple Variants**: 6 padrões diferentes de código (Variables, Functions, Dataclasses, etc).
- **Glassmorphism UI**: Interface premium com efeitos de desfoque e transparência.
- **Ready for Render**: Configuração nativa para deploy estático.

## ⚡ Quick Start

### Requisitos
- Apenas um navegador ou qualquer servidor estático (Python/Node).

### Desenvolvimento Local
```bash
# Clone o repositório
git clone https://github.com/NEO-PROTOCOL/python-2u.git

# Inicie o servidor via Makefile
make dev
```

## 🛠 Tech Stack

- **Core**: HTML5 Semantic / Vanilla JavaScript
- **Styling**: Vanilla CSS (Custom Variables / Glassmorphism)
- **Deployment**: Render.com (Static Site)
- **Automation**: Makefile

## 🏗 Estrutura do Projeto

```text
├── index.html       # Entry point do APP
├── Makefile         # Automação de tarefas
├── render.yaml      # Configuração de Infra (Render.com)
├── README.md        # Documentação NEØ Standard
└── images/          # Ativos visuais
```

## 🐍 Makefile Commands

| Comando | Descrição |
| :--- | :--- |
| `make dev` | Inicia o servidor local em `localhost:8000` |
| `make build` | Prepara os arquivos para distribuição |
| `make clean` | Remove arquivos temporários e cache |
| `make deploy` | Executa o protocolo de Push Seguro NΞØ |
