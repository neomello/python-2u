## рҹҺҜ What is this?

A minimal, futuristic interface that converts any human sentence into Python code вҖ” in real time. Type a feeling, a thought, or a caption idea. Get back **6 distinct Python representations** of that same phrase, formatted with syntax highlighting and ready to copy.

No dependencies. No build step. No backend. Just a `.html` file with a soul.

Built under **NОһГҳ Protocol** standards вҖ” combining Web2 glassmorphism inputs with Web3 terminal outputs, and an animated boot sequence that makes every session feel like a deployment of reality.

## рҹҡҖ Quick Start

Ensure you have **Python 3** installed for the local dev server.

```bash
# Clone the repository
git clone https://github.com/NEO-PROTOCOL/python-2u.git
cd python-2u

# Start local server (localhost:8000)
make dev
```

### Build for Distribution

To generate a clean `dist/` folder for deployment:

```bash
make build
```

## рҹ§¬ Code Variants

When you type `"coffee is fuel"`, the app yields:

- **01 В· Variable**
  в””в”Җ Standard assignment with f-string printing.
- **02 В· Function**
  в””в”Җ Typed function returning an upper-case string.
- **03 В· Dict / Enum**
  в””в”Җ Mapping words to numeric indices (vibe check).
- **04 В· Try / Except**
  в””в”Җ Error handling wrapper for the phrase action.
- **05 В· List Comprehension**
  в””в”Җ Filtering words into a logic list.
- **06 В· Dataclass**
  в””в”Җ Heavyweight structure representing the "Momento".
